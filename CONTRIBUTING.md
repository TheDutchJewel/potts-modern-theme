# Contributing

Bug reports and focused pull requests are welcome through the public GitHub repository.

Before submitting a change:

1. Test with a supported webtrees 2.2.x release.
2. Test as an anonymous visitor and as an administrator.
3. Check desktop and mobile layouts.
4. Run PHP and JavaScript syntax checks.
5. Avoid changes to webtrees core files.
6. Do not include private GEDCOM data, personal photographs, credentials or server paths.

Keep changes narrowly scoped and explain any third-party module dependency.
